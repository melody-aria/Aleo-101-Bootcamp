# Aleo 指令 and snarkVM

欢迎来到 Aleo 指令指南。Aleo 指令是 Aleo 程序的中间表示。 所有 [Leo 程序](<https://docs.leo-lang.org/leo>) 都会编译为 Aleo 指令，再编译为字节码。 如果您的目标是细粒度的电路设计，或者 如果您正在实现一个读取除 Leo 之外的高级语言的编译器，并希望您的程序在 Aleo 上运行，我们建议学习和使用 Aleo 指令。

Aleo 程序是扩展名为 `.aleo` 的文件。 Aleo 程序包含 Aleo 指令——一种类似汇编的编程语言。 Aleo 指令被编译为 AVM 操作码，可由 Aleo 虚拟机执行。


::: info[信息]
snarkVM 目前正在积极开发中。请关注以下仓库： [**GitHub**](<https://github.com/AleoNet/snarkVM>) 以获取可能的破坏性变更。
:::
