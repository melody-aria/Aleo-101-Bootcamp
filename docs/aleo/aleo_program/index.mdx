# Aleo Programs

An **Aleo program** defines private application logic executed by the Aleo VM. Most developers write programs in **Leo** and compile them for the VM.

## 结构

  * **程序 ID** : `program <name>.aleo;`
  * **入口点** : 类型化输入 → 计算 → 类型化输出
  * **类型** : 为基础类型，加上 `struct` 和 `record`




    program foo.aleo;

    function hello:
        input r0 as u32.public;
        input r1 as u32.private;
        add r0 r1 into r2;
        output r2 as u32.private;

## 寄存器与可见性

值存储在**寄存器** （`r0`、`r1`、...）中。每个值都有一个**类型** 和一个**可见性** ：

  * **`.public`** ：公开
  * **`.private`** ：已证明，不公开


## 类型

基础类型包括：


    address
    boolean
    field
    group
    i8
    i16
    i32
    i64
    i128
    u8

    u128

## `struct`


    struct array3:
        a0 as u32;
        a1 as u32;
        a2 as u32;

## `record`（隐私状态）

记录是用户拥有的状态。交易通常**消费** 记录并**创建** 新记录。


    record token:
        owner as address.private;
        amount as u64.private;

## 示例：铸造 + 转账


    program foo.aleo;

    record token:
        owner as address.private;
        amount as u64.private;

    function mint:
        input r0 as u64.private;
        cast self.signer r0 into r1 as token.record;
        output r1 as token.record;

    function transfer_amount:
        input r0 as token.record;
        input r1 as address.private;
        input r2 as u64.private;

        sub r0.amount r2 into r3;
        cast r0.owner r3 into r4 as token.record;
        cast r1 r2 into r5 as token.record;
        output r4 as token.record;
        output r5 as token.record;
