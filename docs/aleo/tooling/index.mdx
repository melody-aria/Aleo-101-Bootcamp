# 开发工具 | Aleo Instructions


::: info[信息]
If you have installed a Leo syntax [plugin](<https://docs.leo-lang.org/getting_started/ide#plugins>). 那么您应该已经能够看到 `.aleo` Aleo 指令的语法高亮。
:::

Aleo 在不同的平台上维护了多个语法高亮实现。如果您在此列表中没有看到您常用的编辑器，请在 [GitHub](<https://github.com/ProvableHQ/leo>) 上提交反馈。

  1. [Sublime Text](#sublime-text)
  2. [Visual Studio Code](#vscode)
  3. [Intellij](#intellij)


## Sublime Text

![](../../../assets/images/sublime-6e349f2b35d3a04d3a294584608c17b1.png)
在此处下载编辑器： <https://www.sublimetext.com/download>。 Aleo 指令对 Sublime LSP 插件的支持通过语言服务器提供。

### 安装

  1. 从 Package Control 安装 [LSP](<https://packagecontrol.io/packages/LSP>) 和 [LSP-leo](<https://packagecontrol.io/packages/LSP-leo>)。
  2. 重启 Sublime。


### 使用方法

按照以下步骤切换 `Aleo 指令` 语法高亮。

  1. 打开 `Sublime Text`。
  2. 从 Settings > Select Color Scheme... > LSP-leo
  3. 这也将允许您看到 Aleo 指令的语法高亮。


## VSCode

![](../../../assets/images/vscode-7c495a48801cb76d14122635d55f4145.png) 在此处下载编辑器： <https://code.visualstudio.com/download>。

### 安装

  1. 从 VSCode 市场安装 [Leo for VSCode](<https://marketplace.visualstudio.com/items?itemName=aleohq.leo-extension>)。
  2. 正确的扩展 ID 是 `provablehq.leo-extension`，描述应显示 "Leo 官方 VSCode 扩展"。


### 使用方法

  1. 打开 `VSCode`。
  2. 前往 Settings > Extensions 或使用左侧面板的 Extensions 按钮启用 Leo 插件。
  3. 这也将允许您看到 Aleo 指令的语法高亮。


## IntelliJ

![](../../../assets/images/intellij-573fa5194d1749d06ab4e7df29eb64de.png) 在此处下载编辑器： <https://www.jetbrains.com/idea/download/>。

### 安装

  1. 从 JetBrains 市场下载 [Aleo Developer Plugin](<https://plugins.jetbrains.com/plugin/19890-aleo-developer>)。
  2. 点击右上角的齿轮图标 > 插件 > 顶部齿轮图标 > 从磁盘安装插件 > 选择下载的 zip 文件
  3. 这也将允许您看到 Aleo 指令的语法高亮。
