# Installation

按以下步骤安装工具，以便在本地开发和测试程序：

## 1\. 安装 Leo 🦁

### 1.1 安装 Git

**[bit.ly/start-git](<https://bit.ly/start-git>)**

### 1.2 安装 Rust

**[bit.ly/start-rust](<https://bit.ly/start-rust>)**

### 1.3 安装 Leo

  * Cargo（推荐）
  * 预编译二进制文件
  * 从源代码构建




    cargo install leo-lang

这会将 `leo` 可执行文件安装到 `~/.cargo/bin/leo`。

从以下地址下载最新版本： [Leo GitHub 发布页面](<https://github.com/ProvableHQ/leo/releases>)，然后：


    # 将二进制文件设为可执行
    chmod +x leo

    # 将其移动到系统路径
    mv leo /usr/local/bin

Pre-built binaries are available for macOS Apple Silicon and other platforms. Browse all options on the [releases page](<https://github.com/ProvableHQ/leo/releases>).


    # Download the source code
    git clone https://github.com/ProvableHQ/leo
    cd leo

    # Install 'leo'
    cargo install --path .

### 1.4 验证安装

要验证安装，请运行：


    leo --version

### 1.5 更新 Leo

您可以使用以下命令将 Leo 更新到最新版本：


    leo update

这份 [Leo CLI 指南](<https://docs.leo-lang.org/cli/overview>) 提供了所有 CLI 命令的说明。

这份 [Leo 语言指南](<https://docs.leo-lang.org/language/overview>) 提供了 Leo 编程语言语法和语义的概述。

### 1.6 可选：IDE 语法高亮

Aleo 为多种流行的代码编辑器维护了语法高亮实现。这可以显著提升您的开发体验，通过提供视觉提示使您的 Leo 代码更易于阅读。以下是支持的编辑器概览：

  1. Visual Studio Code
  2. Sublime Text
  3. Intellij


有关如何为这些编辑器设置语法高亮的详细说明，请参阅 [Leo 开发工具](<https://docs.leo-lang.org/getting_started/ide#plugins>) 指南。

## 2\. 安装 snarkOS

### 2.1 前置条件

**支持的操作系统（仅限 64 位）：**

  * Ubuntu 22.04 LTS
  * macOS Ventura 或更高版本
  * Windows 11 或更高版本


**Rust：** 安装以下文件中指定的 Rust 版本： [`rust-toolchain`](<https://github.com/ProvableHQ/snarkOS/blob/mainnet/rust-toolchain>) 文件，通过 [rustup](<https://www.rust-lang.org/tools/install>).

**【Windows 用户】** 需要额外的依赖项：

  1. 通过 Visual Studio Installer 安装适用于 Windows 的 C++ Clang 工具
  2. 将 `LIBCLANG_PATH` 环境变量设置为已安装的 `libclang.dll` 文件的位置，通常位于：

         Program Files (x86)\Microsoft Visual Studio\2022\BuildTools\VC\Tools\Llvm\x64\bin


### 2.2 克隆 snarkOS 仓库


    git clone --branch mainnet --single-branch https://github.com/ProvableHQ/snarkOS.git
    cd snarkOS

**【Ubuntu 用户】** 提供了一个安装依赖项的辅助脚本：


    ./build_ubuntu.sh

### 2.3 安装 snarkOS


    cargo install --locked --path .

**可选：GPU 支持（实验性）**

CUDA 支持可用但被视为不稳定：


    cargo install --locked --path . --features cuda

### 2.4 验证安装


    snarkos --version
