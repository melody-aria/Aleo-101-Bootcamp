## 开发工具

### 编写 Aleo 程序

以下工具允许开发者为 Aleo 网络编写隐私程序。

名称| 描述
---|---
[Aleo 指令](../../aleo/aleo/index.mdx)| Aleo 协议支持的低级汇编语言，用于编写 Aleo 程序。
[Leo](<https://docs.leo-lang.org/leo>)| 对开发者友好的编程语言，用于在 Aleo 上编写隐私程序。
[Leo Playground](<https://play.leo-lang.org/>)| 用于编写和测试 Leo 程序的基于浏览器的 IDE。

### 用于创建隐私应用的 SDK

以下 SDK 可用于在 Aleo 上构建隐私 Web 和全栈应用。

语言| SDK| 用例
---|---|---
Rust| [snarkVM](<https://github.com/ProvableHQ/snarkVM>)| 基于 Rust 的 Aleo 应用
TypeScript/Wasm| [Provable SDK](<https://docs.explorer.provable.com/docs/sdk/92sd7hgph3ggt-overview>)| Node.JS Aleo 应用与 Aleo 浏览器扩展
TypeScript| [Puzzle SDK](<https://docs.puzzle.online/>)| Aleo 浏览器 Dapp
