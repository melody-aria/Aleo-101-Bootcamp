# 如何获取帮助

Discord 是我们与社区沟通的主要渠道. If you have a question that is not covered in this developer documentation, join our [Discord](<https://discord.com/invite/aleo>) and visit the engineering support channels for help. 你也可以参加社区会议，会议对所有人开放并在 Discord 上举行. 要找到会议的确切日期和时间，请查看我们 Discord 服务器中的"活动"栏目. 目前有 2 个会议:

  * **开发者会议** : 每月第一个和第三个周一举行的月度会议 太平洋时间上午 7:30 - 8:30 供开发者讨论技术主题和更新.
  * **社区会议** : 每月最后一个周一举行的月度会议 太平洋时间上午 8:00 - 9:00 供普通社区成员了解 Aleo 生态系统的最新发展.
